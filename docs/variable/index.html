    public Botao: * source 

    import Botao from 'controle_montagem/src/Components/Botao/Botao.js'

    Componente de Botão. Este componente de botão possui vários estilos personalizáveis através das props. O botão também possui um comportamento de animação ao passar o mouse e ao clicar:

    • Ao passar o mouse: Aumenta ligeiramente de tamanho (scale(1.05)).
    • Ao clicar: Move-se ligeiramente para baixo (translateY(2px)).

    Properties:

    NameTypeAttributeDescription
    props.color string

    Cor do texto. Valor padrão : 'black'
    props.backgroundcolor string

    Cor de fundo do botão. Valor padrão : 'white'
    props.border_radius string

    Raio da curvatura dos cantos do componente. Valor padrão : '0px'
    props.width string

    Comprimento do componente. Valor padrão : '100%'
    props.border string

    Borda do componente. Valor padrão : '0px'
    props.padding string

    Espaçamento interno do componente. Valor padrão : '0px 0px'
    props.margin string

    Espaçamento externo do componente. Valor padrão : '0px 0px'
    props.font_size string

    Tamanho da fonte. Valor padrão : '14px'
    props.boxshadow string

    Sombra do componente. Valor padrão : '0px 0px 0px'

    Example:

    const handleClick = () => {
  alert('Botão Clicado !!');
};
return (
  <Botao 
    color="white" 
    backgroundcolor="blue" 
    border_radius="5px" 
    width="200px" 
    border="1px solid black" 
    padding="10px 20px" 
    margin="10px" 
    font_size="16px" 
    boxshadow="2px 2px 5px rgba(0,0,0,0.3)"
    onClick={handleClick}
  >
    Clique Aqui
  </Botao>
);
